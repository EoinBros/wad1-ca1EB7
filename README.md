# wad1-labs-2026

This project is a simple web application for showcasing a clothing collection. Users can browse a catalogue of items with images, view details in the collection page, and get in touch via a contact form. The interface uses Express with Handlebars templates, styled with custom CSS, and demonstrates client‑side interactions such as popup alerts. The app is built as an assignment for a web app development course and includes features learned from labs 1 & 2.
